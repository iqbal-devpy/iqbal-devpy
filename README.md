## 🛠️ What I Actually Ship

### 🔍 RAG Pipelines  End to End
Ingestion → chunking → embeddings → vector retrieval (**pgvector**, **MongoDB Atlas**) → and the part most people skip: **retrieval quality evaluation and relevance tuning**.

### 🤖 Agentic Workflows with Function Calling
Multi-tool agents grounded in **live data** (Shopify GraphQL, internal APIs) using **LangChain**, **LangGraph**, **CrewAI**, and **AutoGen**  built so every reply is evidence-based, not hallucinated.

### 🔀 Multi-Provider LLM Orchestration
Routing across **GPT**, **Llama**, **Qwen**, and **DeepSeek** (via OpenRouter) on cost/latency/capability trade-offs  with **circuit breakers** and **automatic failover** when providers go down.

### 📊 Evaluation Discipline
Curated test sets, scoring rubrics, and regression checks gating **every** prompt or model change before it ships. *No vibes-based prompt engineering.*

### 🛡️ Production Reliability
Schema-constrained structured outputs · multimodal moderation pipelines · rate limiting · exactly-once processing · guardrails that hold under real traffic.
